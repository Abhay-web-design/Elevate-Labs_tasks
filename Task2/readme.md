# 📝 To-Do Dashboard Web App

A clean and minimal **To-Do Dashboard** built using **HTML, CSS, and JavaScript**.
This project allows users to manage tasks efficiently with a simple and intuitive interface.

---

## 🚀 Features

* ✅ Add new tasks
* 📋 View tasks in **To-Do** and **Completed** sections
* 🔍 Click on any task to view full details
* 🔄 Automatic UI update when task status changes
* 🔙 Back button to navigate from task detail to dashboard

---

## 🖥️ Project Structure

```
📁 project-folder
├── index.html     # Main HTML structure
├── style.css      # Styling and layout
└── script.js      # Application logic
```

---

## ⚙️ How It Works

### 1. Dashboard View

* Displays two sections:

  * **To-Do Tasks**
  * **Completed Tasks**
* Clicking a task opens the detailed view.

### 2. Task Detail View

* Shows:

  * Task title
  * Description
* Includes a **Back button** to return to dashboard.

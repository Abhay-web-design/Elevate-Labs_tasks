# 🍞 Premium Bakery Landing Page

## 📌 Overview

This project is a **modern, responsive bakery landing page** built using **HTML, CSS, and JavaScript**.
It features a premium UI design with smooth animations, glassmorphism effects, and a fully responsive layout.

The website showcases a bakery brand with sections like hero, features, gallery, visit, and about — designed to look like a real-world production website.

---

## 🚀 Features

* ✨ Premium modern UI design
* 📱 Fully responsive (mobile + tablet + desktop)
* 🍔 Mobile navbar with hamburger menu
* 🎨 Glassmorphism effects (blur overlays)
* 🖱 Smooth hover animations
* 🖼 Image-based sections (hero & visit)
* 📸 Gallery section
* 📖 About section with storytelling
* 🔘 Call-to-action buttons

---

## 🛠️ Technologies Used

* **HTML5** – Structure of the webpage
* **CSS3** – Styling, layout, animations
* **JavaScript (Vanilla)** – Navbar toggle functionality
* **Google Fonts** – Typography (Playfair Display & Poppins)

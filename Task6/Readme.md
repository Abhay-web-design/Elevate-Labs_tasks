# 📩 Contact Form with Validation

## 📌 Project Overview
This project is a **responsive contact form** built using HTML, CSS, and JavaScript.  
It includes **client-side validation** to ensure users enter valid data before submitting the form.

---

## 🚀 Features
- ✅ Responsive design (mobile, tablet, desktop)
- ✅ Clean and modern UI
- ✅ Input validation using JavaScript
- ✅ Email format validation using Regex
- ✅ Error messages for invalid inputs
- ✅ Success message on valid submission
- ✅ Prevents form submission if inputs are invalid

---

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla JS)

---

## 📂 Project Structure
├── index.html
├── style.css
└── script.js

---

## ▶️ How to Run
1. Download or clone the repository  
2. Open `index.html` in any browser  

---

## 🧠 Concepts Used
- Form Elements
- DOM Manipulation
- Event Handling
- Client-side Validation
- Regular Expressions (Regex)

---

## ❓ Validation Rules
- Name → Cannot be empty  
- Email → Must be in valid format (example: abc@gmail.com)  
- Message → Cannot be empty
